# portfolio（開発中）
このアプリは、自分の冷蔵庫にある食材を記録・管理するためのシンプルなWebアプリです。  
React（Vite）とNode.js（Express）、MongoDBを使って作っています。 

## できたこと

- ユーザー登録・ログイン（JWT認証）
- 食材のCRUD

---

## 技術スタック

### フロントエンド

- React（Vite）

### バックエンド

- Node.js（Express）
- MongoDB（Mongoose）
- JWT（ログイン認証）
- passport-local-mongoose
